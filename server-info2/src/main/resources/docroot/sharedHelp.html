This application demonstrates Mule runtime object stores.

You can test various object store operations

  • Store key/value pairs in the object store

    • http://localhost:#[flowVars.httpPort]/store?key=someKey&value=someValue

    • http://localhost:#[flowVars.httpPort]/store?key=Alice&value=first
    • http://localhost:#[flowVars.httpPort]/store?key=Bob&value=second
      Result: Will list the current array of all keys and values in the Object store.
      The result will indicate if this is the first time the key is added to the object store, or if the key already exists in the objects and the new value is updated.

  • Retrieve a value by key

    • http://localhost:#[flowVars.httpPort]/retrieve?key=someKey

    • http://localhost:#[flowVars.httpPort]/retrieve?key=Alice
    • http://localhost:#[flowVars.httpPort]/retrieve?key=Bob
      Result: Returns the value for this key.
